# mi-proyecto

Login

La página de inicio de sesión sirve para que el usuario entre con su cuenta. Pone usuario y contraseña, y la función logear() compara esos datos con lo que está guardado en el localStorage.
Si coinciden, lo deja pasar y lo manda a la página principal. Si están mal, muestra un mensaje de error. Es un login simple pero funciona bien.


Página principal con el juego 

La página principal solo se abre si el usuario está logueado. Tiene un botón para cerrar sesión, un minijuego de serpiente y chistes sobre Chucky Norri. El Snake se controla con las flechas y si chocás aparece un mensaje de que perdiste. El botón de cerrar sesión borra el login y te manda al inicio.
